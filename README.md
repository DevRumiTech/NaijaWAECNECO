NaijaWAECNECO

A simple WAEC & NECO grade calculator built for Nigerian students.

Users can input their subjects and grades to instantly see total points,
average score, and a clear performance interpretation.

This is a static website built with HTML, CSS, and vanilla JavaScript.
No data is collected.

Live site: https://devrumitech.github.io/NaijaWAECNECO/
